Obtaining descriptive statistics into an output dataset

Question Answer
How do we specify the input dataset for PROC MEANS? Use the `data=` option on the PROC MEANS statement.
What happens if we do not provide the data= option? The most recently created dataset in the current SAS session is used.
How do we specify the variables for which statistics are requested? Use the `var` statement.
What happens if we do not provide a VAR statement? SAS produces descriptive statistics for all numeric variables in the input dataset.
How do we specify the output dataset name? Use the `output` statement with the `out=` option.
What happens if we do not provide the output statement? No output dataset is created, but summary statistics are still shown in the output window.

Create a dataset named stats01 to store the default descriptive statistics of all numeric variables of class01

  • We specify the input dataset with `data=class01`
  • We specify the output dataset with `output out=stats01;`
  • Because no `var` statement is used, PROC MEANS analyses all numeric variables
  • The output dataset contains automatic variables such as `_TYPE_`, `_FREQ_`, and `_STAT_`
  • By default, PROC MEANS produces rows for N, Mean, Std Dev, Minimum, and Maximum
SAS Log
  • Inspect `stats01` and confirm that each row corresponds to one default statistic
  • `_STAT_` identifies the statistic represented on that row

Create a dataset named stats04 to store N, Mean, and Median of height

  • Specific statistics are requested in the `output` statement
  • Each statistic keyword is followed by an equal sign and the name of the output variable that should store the result
  • When statistics are named explicitly, the `_STAT_` variable is no longer needed
SAS Log
Dataset View

Create a dataset named stats05 to store N, Mean, and Median of height and weight

  • When more than one analysis variable is listed, the output variable names after each statistic keyword are assigned in the same order
  • So `height_n` stores N for height and `weight_n` stores N for weight
  • The same ordering rule applies to the mean and median output variables
SAS Log
Dataset View

Create a dataset named stats06 to store N, Mean, and Median of age, height, and weight

  • This example shows that the output variable names do not have to match the input variable names
  • What matters is the position of each output variable name after the statistic keyword
  • The first name corresponds to `age`, the second to `height`, and the third to `weight`
SAS Log
  • In this example `a101` stores the mean of height because height is the second variable on the VAR statement
  • `x101` stores the median of weight because weight is the third variable on the VAR statement
Dataset View

Create a dataset named stats07 using the AUTONAME option

  • If the number of variables is large, manually naming every output variable can be tedious
  • The `autoname` option tells SAS to generate output variable names automatically
  • SAS combines the analysis variable name and statistic name to create descriptive output names

Create a dataset named stats08 to store N, Mean, and Std of height within each age group

  • When statistics are needed within groups, we use BY group processing
  • To use a BY statement, the input dataset must first be sorted by the same BY variable
  • The output dataset will contain one row per BY group when statistics are explicitly named
SAS Log
  • Inspect `stats08` and confirm that each age value has its own summary row
  • The variable `age` is carried into the output dataset because it is the BY variable
Dataset View

Create a dataset named stats09 to store N, Mean, and Std of height within each sex group using CLASS

  • The `class` statement provides grouped summaries without requiring the data to be presorted
  • Unlike BY processing, CLASS can also produce overall summary rows together with class-level rows
  • In the output dataset, `_TYPE_` helps identify the summary level represented by each row
SAS Log
  • Review `stats09` and compare it with `stats08` to understand the difference between BY and CLASS processing
Dataset View

Key points to remember

  • `proc means` calculates descriptive statistics for numeric variables
  • The `var` statement controls which numeric variables are analysed
  • The `output` statement creates a SAS dataset containing the results
  • `autoname` is useful when many output statistic variables must be created
  • BY and CLASS both support grouped statistics, but they behave differently
