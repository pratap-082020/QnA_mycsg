001 General Electrocardiogram information is collected in ecg dataset. So, this dataset becomes the base input dataset for creating SDTM.EG dataset. The information related to the tests performed are collected as variables in the input datasets. As the expected structure for SDTM EG domain is one record per subject per test per timepoint, we need to transform the test results and other dependent variables from horizontal to vertical structure.
002 STUDYID Study Identifier raw.ecg.STUDY
003 DOMAIN Domain Abbreviation Set to "EG"

005 EGSEQ Sequence Number Sort the final dataset by USUBJID, EGCAT, EGTESTCD, EGDTC and set to 1 on the first record of the subject and increment by 1 on subsequent records.

Set to "QRS" for the rows representing result from raw.ecg.QRS_RAW

016 EGBLFL Baseline Flag Set to "Y" on the latest record with non-missing result on or prior to SDTM.DM.RFSTDTC per EGTESTCD.
017 VISITNUM Visit Number Set to 1 when VISIT="SCREENING", Else when VISIT contains "WEEK" then add 100 to the week number (that is, WEEK 1 will have a VISITNUM of 101). Else when VISIT contains "FOLLOW-UP" then add 200 to the numeric portion of the VISIT. For Unscheduled visits, add the unscheduled visitnumber as a two digit decimal to the previous scheduled visit number. (For example, a third unscheduled visit after VISITNUM 102 will be formatted as 102.03)
018 VISIT Visit Name Set to "SCREENING" when FOLDER="SCR". Else when FOLDER contains "Week" then convert FOLDER to uppercase.Else when FOLDER contains "FU" then replace "FU" with "FOLLOW-UP". Unscheduled visits (FOLDER contains "UNS_") will be remapped based the previous scheduled VISIT when sorted by VSDTC. For each unscheduled visit present after a scheduled visit, concatenate the word 'UNSCHEDULED' to the previous scheduled visit followed by the number of the unscheduled padded with a leading zero. (For example, a third unscheduled visit after "WEEK 2" will be formatted as 'WEEK 2 UNSCHEDULED 03")
019 EPOCH Epoch Derive by comparing EGDTC with the sdtm.se.SESTDTC and sdtm.se.SEENDTC of each element of a subject and assign the epoch based on the sdtm.se.EPOCH of the element in which each event start date falls.
020 EGDTC Date/Time of ECG Convert raw.ecg.EGDT_RAW to ISO 8601 format.
021 EGDY Study Day of ECG numeric(datepart(sdtm.eg.EGDTC))-numeric(datepart(sdtm.dm.RFSTDTC))+(numeric(datepart(sdtm.eg.EGDTC))>=numeric(datepart(sdtm.dm.RFSTDTC)))
