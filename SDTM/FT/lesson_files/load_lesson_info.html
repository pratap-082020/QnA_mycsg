Background
  • In some clinical trials, we may have to collect data about some task-based evaluations like assessment of mobility, assessment of dexterity, or cognitive ability of subjects
  • There are several stand-alone, validated instruments (forms/questionnaires) which fall under this class of 'task-based' evaluations
  • Some examples are:
    • 10-METER WALK/RUN
    • 4-STAIR ASCEND
    • 4-STAIR DESCEND
    • Six Minute Walk Test
    • PUL (Performance of the Upper Limb Module)
    • SPPB (Short Physical Performance Battery)
    • MMSE (Mini-Mental State Examination)
  • SDTM standard has a standard domain for capturing these findings - "Functional Tests (FT)"
  • FT dataset belongs to Findings class and has an expected record structure of 'One record per Functional Test finding per time point per visit per subject'

What type of information is typically collected or presented?
  • Date of assessment
  • Test performed status
    • Yes
    • No 
      • Reason not performed
  • Short name of the test
  • Name of the test
  • Position of the subject when test was performed
  • Result
    • in original units
    • in standard units
  • Derived information
    • Baseline flag - when results of a test are collected over pre and post treatment period
